Cálculo do imposto de renda + Padrão Strategy

Neste exercício abordarei a implementação do padrão de projeto estratégia (do inglês Strategy) para construir um programa que calcule o valor do imposto pago por uma pessoa, dado seu salário.

O padrão tem a seguinte proposta:

  • Definir uma família de algoritmos, encapsular cada uma delas e torná-las intercambiáveis. Strategy permite que o algoritmo varie independentemente dos clientes que o utilizam

Ao invés de criar um único método que faça todo o cálculo do imposto, criamos classes para calcular o imposto de acordo com o salário. E para encapsular e permitir que futuras alterações possam ser facilmente realizadas, criamos uma interface chamada de CalcImpostoStrategy. Sendo assim, precisamos apenas passar o salário como parâmetro e instanciar a classe que é responsável por calcular aquele valor de salário.
